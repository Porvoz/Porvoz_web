# Porvoz – Recordatorios de Medicación por Llamadas de Voz

Porvoz es un prototipo de aplicación web orientado a apoyar a adultos mayores y pacientes con enfermedades crónicas en la correcta toma de sus medicamentos, mediante llamadas automatizadas con inteligencia artificial que recuerdan dosis y horarios, y permiten a cuidadores hacer seguimiento al cumplimiento del tratamiento.
